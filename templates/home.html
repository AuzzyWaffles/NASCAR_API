Welcome to the NASCAR API!

Use /tracks?state=state to find NASCAR tracks by state
(all tracks will be listed if state is left blank)

Use /season?series=series&year=YYYY to find season schedule by year. Eligible series values: Cup, Xfinity
(series will default to NASCAR Cup Series if left blank, and year will default to current year if left blank).

The admin can use /post/tracks?name=name&length=length&type=type&state=state to post racetracks to the db.

The admin can use /post/races?track_id=track_id&name=name&series=series&date=date&winner=winner to post races in a particular season schedule to the db.
